2023-2024

DZ-Mouhami, a web directory of lawyers in Algeria

During my Licence 3, I worked with my team on "DZ-MOUHAMI," a comprehensive web directory for Algerian lawyers. This project involved developing a platform where users can search for lawyers based on specialization, location, and other criteria. Key features included user registration, detailed lawyer profiles with geographical mapping, advanced search functions, appointment scheduling, and client reviews. The system was built using React for the front end and Python for the back end, with a focus on a responsive design, bilingual support, and fast loading times. The project also included data scraping, unit testing, and automated functional testing with Selenium.
